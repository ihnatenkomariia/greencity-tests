## Test Case 3: Пошук події за назвою

**Preconditions:**
* Користувач знаходиться на сторінці [Events](https://www.greencity.cx.ua/#/greenCity/events).
* Існує подія з назвою "Eco Workshop".

**Test Steps:**

| Step  | Action                           | Data           | Expected Result                                       |
| :--- | :--- | :--- | :--- |
| 1     | Ввести назву події в поле пошуку | "Eco Workshop" | Список подій відфільтровано.                          |
| 2     | Перевірити результат пошуку      | -              | Картка з назвою "Eco Workshop" відображається першою. |

| Step | Action                           | Data                         | Expected Result                                                           |

| 1    | Перевірити заголовок та автора   | -                            | Відображається назва ("kkkhhh..."), дата публікації та ім'я автора        |
